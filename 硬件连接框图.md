## 信号连接明细

### 8255A(并行接口扩展,经 P0 总线)

| 8255A 引脚 | 连接到 | 说明 |
| --- | --- | --- |
| D0~D7 | P0.0~P0.7 | 8 位数据总线 |
| A0 | P2.0 | 端口选择低位 |
| A1 | P2.1 | 端口选择高位 |
| /CS | P2.7 | 高位地址直接译码选通,低 8 位地址不锁存 |
| /WR | P3.6(/WR) | 写选通 |
| /RD | P3.7(/RD) | 读选通 |

地址分配:**PA 口 = 0x7C00,PC 口 = 0x7E00,控制口 = 0x7F00**

### LCD1602(挂在 8255A 上,只写方式)

| LCD1602 引脚 | 连接到 |
| --- | --- |
| D0~D7 | 8255A PA0~PA7 |
| RS | 8255A PC0 |
| E | 8255A PC1 |
| R/W | GND(固定为写) |

### DS1302(实时时钟,三线接口)

| DS1302 引脚 | 连接到 |
| --- | --- |
| I/O | P1.0 |
| SCLK | P1.1 |
| CE | P1.2 |
| X1、X2 | 32.768 kHz 晶振 |

### 24C02(E²PROM,I²C 接口)

| 24C02 引脚 | 连接到 |
| --- | --- |
| SCL | P1.3(经上拉电阻接 VCC) |
| SDA | P1.4(经上拉电阻接 VCC) |
| A0~A2、WP | GND |

### 按键(独立式,另一端接地)

| 按键 | 连接到 |
| --- | --- |
| 清除 | P3.2 |
| 启停 | P3.3 |
| 换挡 | P3.4 |
| 加数 | P3.5 |
