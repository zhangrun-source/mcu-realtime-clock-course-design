### 0.3 三条设计主线

1. **8255 做接口核心**：LCD1602 的 8 位数据走 PA，控制线走 PC，扣住课程主题。
2. **DS1302 做主时钟（全权管时间）**：时分秒 + 年月日全部由 DS1302 提供，走时准、掉电靠电池脚不丢。**启停**用它秒寄存器的 **CH 位**（bit7：1=停走/0=走时），**清除/预置**就是往它的寄存器写值，逻辑统一。
3. **Timer0 只做刷新节拍**：ISR 不再计时，只每 0.1s 置一次 `bRefresh` 刷新标志（满足任务书"定时中断"要求，答辩讲"定时器负责显示刷新调度，DS1302 做高精度时钟源"）。慢速的 LCD 写、DS1302 读、24C02 备份都放主循环——中断轻、主循环重。

## 1. 芯片介绍与作用

### 1.1 8031

- 用 **8031**：8051 系列的**无片内 ROM** 版本，程序必须放在**外部程序存储器**里，所以 **EA 必须接 GND**（低电平，强制从外部取指）。
- **程序存储器**：外挂一片 EPROM（如 27C256/2764），P0 分时送地址低 8 位/数据，需经 **74LS373** 锁存低 8 位地址（由 ALE 触发）；P2 送地址高 8 位。
    - > 注意：这个 74LS373 是给**外部程序存储器**用的，和 §1.2 里 8255 那个"不锁存低 8 位"技巧是两码事——8255 靠 P2 高位选片，本来就不占这条锁存链路。
        
- **Proteus 里的简化做法**：8031 模型可以直接在其 `Program File` 属性里挂上编译好的 `.hex` 就能仿真运行，不接外部 EPROM 也能跑（省事，推荐先这么调）；要做"完整硬件"再补 27C256 + 74LS373 那一支。
- 12MHz 晶振 → 机器周期 1μs，方便算定时。
- 资源分配：P0=数据总线（接 8255）、P2 高位=地址译码、P1=DS1302/24C02 串行线、P3=按键 + 读写控制线。

### 1.2 8255A —— 可编程并行接口（本课重点）

**作用**：给 CPU 扩出三个 8 位并行口（PA/PB/PC），本设计用来接 LCD1602。

**方式与控制字**：全部用**方式0（基本输入输出）**，PA、PB、PC 全设为**输出**。

- 控制字 = `1000 0000B = 0x80`（D7=1 方式设定，PA/PB=方式0输出，PC 高/低=输出）。

**地址译码——任务书的"不锁存低 8 位"技巧**：

- 用 **P2 的高位地址**直接做 `/CS` 和端口选择（A0、A1），这样 P0 上的低 8 位地址就**不需要 74LS373 锁存**。
- 接法：`A0←P2.0`、`A1←P2.1`、`/CS←P2.7`（低有效）、`/RD←P3.7(RD)`、`/WR←P3.6(WR)`。
- 端口地址（用 `MOVX @DPTR`，只看 DPH=P2）：

|端口|A1 A0|P2.7=0 且 ……|地址（惯用写法）|
|---|---|---|---|
|PA（LCD 数据）|0 0|P2.1=0,P2.0=0|`0x7C00`|
|PB（本设计不用）|0 1||`0x7D00`|
|PC（LCD 控制）|1 0||`0x7E00`|
|控制口|1 1||`0x7F00`|

> 这里 `/CS=P2.7`，因为系统没有外部 RAM，不会冲突；写代码时用 `0x7Cxx` 系列即可。

### 1.3 DS1302 —— 实时时钟芯片（扩展 5，本设计的**主时钟**，全权管时间）

**作用**：独立走时（秒/分/时/日/月/星期/年），带 VCC1 备份电池脚，掉电继续跑。本方案**时分秒 + 年月日全部读它**，显示的 `hh:mm:ss` 和 `yy-mm-dd` 都以 DS1302 为准。数据全是 **BCD 码**。

**引脚**：VCC2(主)、VCC1(备份/电池)、GND、X1/X2(接 **32.768kHz** 晶振)、`/RST(CE)`、SCLK、I/O。

- 接线：`CE←P1.0`、`SCLK←P1.1`、`I/O←P1.2`。

**关键寄存器（写地址/读地址，BCD）**：

|项|写|读|备注|
|---|---|---|---|
|秒|0x80|0x81|**bit7=CH**：1=停走 0=走时（**启停就翻转它**）；低7位=秒 BCD|
|分|0x82|0x83||
|时|0x84|0x85|bit7=0 选 24 小时制|
|日|0x86|0x87||
|月|0x88|0x89||
|星期|0x8A|0x8B|可选|
|年|0x8C|0x8D|00–99|
|WP|0x8E|0x8F|**bit7=写保护**，写任何寄存器前必须先写 0x00 解锁|

> 首次仿真时先写一次 DS1302，把时分秒+年月日设成初值（写前先解 WP，写完可再置回），之后正常只读、按键操作时才写。

**三线时序（要点，细节你实现）**：

- CE 拉高开始通信；先发 1 字节命令（bit7=1, bit0: 1=读/0=写），再发/收数据；**都是 LSB 先出**。
- 写：数据在 SCLK **上升沿**被 DS1302 锁存；读：数据在 SCLK **下降沿**后读 I/O。
- 通信完 CE 拉低。

### 1.4 24C02 —— I²C EEPROM（扩展 10，掉电状态备份）

**作用**：256 字节非易失存储。DS1302 只存"时间"，**应用层状态**（运行/暂停、当前模式、预置配置、心跳计数）存这里，掉电重启后读回恢复。

**引脚**：A2/A1/A0（全接 GND → 器件地址高位固定）、SDA、SCL、WP（接 GND 允许写）、VCC、GND。

- 接线：`SCL←P1.3`、`SDA←P1.4`。
- 器件地址：写 `0xA0`、读 `0xA1`。

**I²C 时序（软件模拟，要点）**：

- **起始**：SCL 高时 SDA 高→低；**停止**：SCL 高时 SDA 低→高。
- **应答(ACK)**：第 9 个时钟从机把 SDA 拉低。
- **字节写**：Start→0xA0→ack→字节地址→ack→数据→ack→Stop，之后**等 5ms** 写周期。
- **随机读**：Start→0xA0→ack→地址→ack→重新 Start→0xA1→ack→读数据→**主机发 NACK**→Stop。

### 1.5 LCD1602 —— 字符液晶（扩展 7，显示）

**作用**：16×2 字符，显示 `TIME 00:00:00` 与 `DATE 26-07-08`。

**接法（经 8255）**：`D0~D7←PA0~PA7`、`RS←PC0`、`E←PC1`、**RW 直接接 GND**（只写不读，用固定延时代替忙检测，最省事）。VO 用电位器调对比度。

**常用指令**：

|指令|码|含义|
|---|---|---|
|功能设置|0x38|8位/2行/5×8|
|显示开关|0x0C|开显示、关光标|
|输入模式|0x06|地址自增、画面不移|
|清屏|0x01|清屏（需 >1.5ms）|
|设地址|0x80\|addr|第1行 0x00 起，第2行 0x40 起|

**写时序**：RS 选指令(0)/数据(1) → PA 送字节 → E 拉高再拉低（RW=0）。

### 1.6 按键（4 个）

- 清除、启停、换挡、加数，接 `P3.2 / P3.3 / P3.4 / P3.5`，各带上拉，按下为低。
- **必须去抖**（检测到低电平后延时 10–20ms 再确认），并做"松手才算一次"防连发。

### 2.2 Proteus 注意点

- SDA、SCL 各接一个 **4.7k 上拉电阻**到 VCC（I²C 必需）。
- DS1302 的 32.768kHz 晶振别用错频率，否则走时不准。
- LCD1602 记得接 VO 对比度电位器，否则仿真时"全黑"或"全空"以为程序错了。
- 按键那 4 个脚开内部/外部上拉；仿真里可直接用带上拉的按钮。
- **8031 上程序**：Proteus 里双击 8031 → 在 `Program File` 里挂上 Keil 编译出的 `.hex` 即可仿真；先不接外部 EPROM，等基本功能通了再决定要不要补 27C256+74LS373 那一支（EA 记得接 GND）。

## 3. 软件架构与算法原理

### 3.1 分层

```
显示层   : LCD 初始化 / 写指令 / 写数据 / 定位 / 显示时间行、日期行
逻辑层   : 主状态机（正常显示 / 预置模式）、按键处理、上电恢复
时基层   : DS1302 读时分秒+年月日 / 写寄存器(校时/清除/预置) / 置CH位(启停)
存储层   : 24C02 读/写（备份与恢复）
底层驱动 : 8255 访问宏、DS1302 三线时序、软件I2C、us级延时
系统节拍 : Timer0 50ms 中断 → 置刷新标志
```

### 3.2 时基与节拍（核心思想）

- **DS1302 是唯一权威时间源**。时分秒、年月日都从它读，8031 不自己数秒。
- **Timer0 定时中断**只做一件事：每 0.1s 置一次 `bRefresh=1`（刷新节拍），**不做计时、不写 LCD、不读 DS1302/24C02**（那些慢，放主循环）。
- 主循环：`if(bRefresh){ 读DS1302 时分秒+年月日 → 组织显示缓冲 → 刷新LCD → bRefresh=0; }`，其余时间轮询按键。
- **启停** = 翻转 DS1302 秒寄存器的 **CH 位**（读 0x81 → 改 bit7 → 写 0x80）。CH=1 芯片停走、显示定住；CH=0 继续。
- **原软件计时的 `g_*`、`bRun` 不再用于走时**——阶段②写的那套可以保留当学习记录，但显示改成读 DS1302。

> Timer0 0.1s @6MHz：机器周期 2μs，计数 50000，初值 = 65536−50000 = 15536 = `0x3CB0`（`TH0=0x3C, TL0=0xB0`, `TMOD=0x01`）。ISR 里 `tick` 累计 **10 次 = 1 秒**（本方案其实不用数满 1 秒，每 0.1s 刷新一次显示反而更跟手）。

### 3.3 功能算法

**扩展(1) 清除 / 启停**（操作 DS1302 寄存器）

- **清除**：往 DS1302 写 时=00、分=00、秒=00（写前先解 WP，秒的 CH 位保持 0=走），回到 `00:00:00` 重新走。
- **启停**：读秒寄存器(0x81) → 翻转 **bit7(CH)** → 写回(0x80)。CH=1 停走、显示定住；CH=0 继续。每按一次翻转一次。

**扩展(2) 预置**（写 DS1302 寄存器）

- 进入预置模式（"换挡"键触发）。建议先置 CH=1 停走，避免边设边跳。
- "换挡"键在 时→分→秒（→退出）之间轮换，用光标/闪烁标出当前编辑位。
- "加数"键让当前位 +1（时 0–23、分/秒 0–59 循环），确认后把值（转成 **BCD**）写进 DS1302 对应寄存器。
- 退出预置时清 CH=0 恢复走时。建议 `enum {NORMAL, SET_HOUR, SET_MIN, SET_SEC}` 状态机管理。
- ⚠️ 写 DS1302 前务必先 `0x8E←0x00` 解写保护，写完可再 `0x8E←0x80` 上锁。

**扩展(5) DS1302 时间 + 年月日**

- 第 1 行 `TIME hh:mm:ss`、第 2 行 `DATE yy-mm-dd`（可加星期），**全部读 DS1302**。读回是 BCD → 拆高低半字节（`>>4` / `&0x0F`）→ `+'0'` 转 ASCII 送 LCD。

**扩展(7) LCD1602**：见 3.4。

**扩展(10) 24C02 备份/恢复**

- DS1302 有电池脚本就掉电不丢时间，所以 24C02 备份的是**应用层状态**（不是时间本身）：
    - `addr0 = 魔术字 0x5A`（判断是否存过有效数据）
    - `addr1 = 当前模式 / 运行暂停态(CH)`
    - `addr2 = 预置配置 / 心跳计数`（演示"实时备份"）
- **上电流程**：读 addr0，若=0x5A → 读回状态恢复；否则首次上电，写默认值。
- **备份时机**：状态变化时或每秒写一次（24C02 写周期 5ms、有擦写寿命，别刷太狠）。→ 对应任务书"掉电重加电后从故障时刻状态恢复"。

### 3.4 LCD 显示流程

```
上电 → 延时>15ms → 写0x38 → 写0x0C → 写0x06 → 写0x01(清屏,>1.5ms)
显示 → 设地址(0x80|pos) → 逐字符写数据(RS=1)
```

### 3.5 主程序总流程

```
main:
  系统初始化(Timer0=0.1s、8255控制字、LCD初始化)
  DS1302: 首次上电检测→若未走时(CH=1)则写一次初值并清CH
  24C02: 读魔术字 → 若=0x5A 恢复状态，否则写默认
  开中断
  while(1):
     扫描按键 → 分发(清除/启停/换挡/加数 → 写DS1302寄存器)
     if bRefresh:
         读DS1302 时分秒 → 显示第1行 hh:mm:ss
         读DS1302 年月日 → 显示第2行 yy-mm-dd
         bRefresh = 0

T0中断:
     重装初值; bRefresh = 1;   // 0.1s 一次，只置刷新标志
```

## 5. 分阶段实施计划（对照评分）

|阶段|目标|验收标准|
|---|---|---|
|① 显示打通|8031+8255+LCD1602，硬写一串固定字符|LCD 稳定显示 `HELLO`|
|② 时钟跑起来|先用 Timer0 软件计时点亮 `hh:mm:ss`（已完成，作为过渡）|秒跳、进位正常|
|③ DS1302 接管|读 DS1302 时分秒→第1行、年月日→第2行，替换软件计时|两行都对、掉电不丢 → 扩展(5) 完成|
|④ 清除/启停|两个键：写00清零、翻转 CH 停走/继续|扩展(1) 完成（→ 中等门槛之一）|
|⑤ 预置|换挡+加数 设置时分秒并写回|扩展(2) 完成（→ **达到中等**）|
|⑥ 24C02 备份|状态存 EEPROM，断电再上电恢复|扩展(10) 完成（→ **达到良好**）|
|⑦ 报告+示波器|虚拟示波器测 SCLK/E 等关键信号并分析|满足任务书"信号测量"，报告完整|

> **强烈建议按此顺序**，每一阶段跑通再进下一阶段。中途出问题时"上一步是好的"能极大缩小排查范围。
